# Project 4: Data Visualization

## Project Overview

This project focuses on **Data Visualization** using Python. The main objective of this task is to create meaningful visual representations from the dataset to communicate insights clearly and effectively.

The dataset contains order-related information such as product name, quantity, unit price, total price, payment method, order status, referral source, coupon code, and order date. Using this data, different charts were created to understand sales performance, revenue trends, customer behavior, payment preferences, and order patterns.

---

## Objective

The objective of this project is to:

- Load and understand the dataset
- Clean basic text and date columns
- Create visual charts from the dataset
- Analyze revenue patterns
- Identify top-performing products
- Understand monthly revenue trends
- Analyze order status distribution
- Compare payment methods
- Highlight important business insights
